Projeto: Calculadora de Desconto em PHP

Tecnologias:

- HTML5
- CSS
- PHP

Funcionalidades:

- Cálculo de desconto percentual
- Processamento de formulário via método POST
- Validação básica de dados
- Exibição dinâmica de mensagens de erro e sucesso

Descrição:

Projeto desenvolvido durante práticas acadêmicas no curso de Análise e Desenvolvimento de Sistemas. O projeto tem como objetivo reforçar conceitos de lógica de programação, estruturas condicionais e integração entre front-end e back-end utilizando PHP.  
O sistema recebe um valor e uma porcentagem de desconto, realiza o cálculo no servidor e exibe o resultado dinamicamente na mesma página.
